# MANUAL MESTRE DO PROJETO V3

Este bundle implementa a base executável do JARVIS CORE V3 com cronologia consolidada, missão formal, RCA, SDDF, Aegis, memória e runtime híbrido.

## Blocos incluídos neste nível
- Mission parser
- Intent resolver
- Project registry
- TechNetGame adapter
- Evidence collector
- Root cause classifier
- Cause chain simplificada
- Gate runner
- Aegis policy engine simplificada
- Ollama client
- Runtime Go inicial
